This project is a work in progress inteneded to start building out some commandable and queryable microcontrollers based on the Arduino platform.
I intend to work with and build different .ino files for different suites of sensors, such as those available from Seeed (grove type) or dfrobot, or other sensors as I find and learn how to program for them.
I hope to have them all controllable from a single controller script, but we'll see how that goes.
